# Diccionario de datos

## Base de datos 1

**Descripción de la tabla o fuente de datos.

| Variable | Descripción | Tipo de dato | Rango/Valores posibles | Fuente de datos |
| --- | --- | --- | --- | --- |
| Age | Edad del paciente | Numérico | Años (aprox. 29–77) | Conjunto de datos cardíacos |
| Sex | Sexo del paciente | Categórico | M: Masculino, F: Femenino | Conjunto de datos cardíacos |
| Chest Pain Type | Tipo de dolor en el pecho | Categórico | TA: Angina Típica, ATA: Angina Atípica, NAP: Dolor No Anginoso, ASY: Asintomático | Conjunto de datos cardíacos |
| Resting BP | Presión arterial en reposo | Numérico | mm Hg (aprox. 94–200) | Conjunto de datos cardíacos |
| Cholesterol | Colesterol sérico | Numérico | mg/dl (aprox. 126–564) | Conjunto de datos cardíacos |
| Fasting BS | Glucosa en ayunas | Binario | 1: >120 mg/dl, 0: ≤120 mg/dl | Conjunto de datos cardíacos |
| Resting ECG | Resultado del electrocardiograma en reposo | Categórico | Normal, ST: Anormalidad en onda ST-T, LVH: Hipertrofia ventricular izquierda | Conjunto de datos cardíacos |
| Max HR | Frecuencia cardíaca máxima alcanzada | Numérico | Entre 60 y 202 latidos/min | Conjunto de datos cardíacos |
| Exercise Angina | Angina inducida por ejercicio | Binario | Y: Sí, N: No | Conjunto de datos cardíacos |
| Old Peak | Depresión del segmento ST inducida por ejercicio | Numérico | Valores decimales (aprox. 0.0–6.2) | Conjunto de datos cardíacos |
| ST Slope | Pendiente del segmento ST durante el ejercicio máximo | Categórico | Up: Ascendente, Flat: Plana, Down: Descendente | Conjunto de datos cardíacos |
| Heart Disease | Indica si el paciente presenta enfermedad cardíaca | Binario | 1: Con enfermedad, 0: Sin enfermedad | Conjunto de datos cardíacos |

- **Variable**: nombre de la variable.
- **Descripción**: breve descripción de la variable.
- **Tipo de dato**: tipo de dato que contiene la variable.
- **Rango/Valores posibles**: rango o valores que puede tomar la variable.
- **Fuente de datos**: fuente de los datos de la variable.
